## Continue 사용 방법

### 1. Chat 열기
```
Ctrl+L (또는 Cmd+L)
```

### 2. 모델 선택
```
상단 드롭다운에서 "Qwen3 Local" 선택
```

### 3. 코드 생성
```
프롬프트 입력 → Enter
또는 Stop (⏹) 버튼으로 중단
```

### 4. 코드 삽입
```
"Insert" 버튼 클릭 → 현재 파일에 추가
```
